# Technohaven - Luxury Website

## About Technohaven

Technohaven is a dual-hub enterprise delivering institutional-grade technology and commercial agility across Southeast Asia and South Asia. This repository contains the luxury brand website showcasing Technohaven's engineering excellence and digital transformation services.

## Features

- **Dual-Hub Architecture**: Bridging legacy engineering (Dhaka) with commercial agility (Kuala Lumpur)
- **Premium User Experience**: High-performance animations and interactive elements
- **Services Showcase**: Project-based and product-based service offerings
- **Case Studies**: Real-world implementations and success stories
- **Contact & Engagement**: Seamless inquiry and connection pathways

## Technology Stack

### Frontend
- React 18+ with TypeScript
- Framer Motion for animations
- GSAP for advanced scroll animations
- Tailwind CSS for styling

### Backend
- Vite for fast build and development

### UI Components
- Radix UI for accessible components
- Custom design system with luxury aesthetics

## Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:5173/`

### Production Build

```bash
npm run build
```

## Project Structure

- `/src/app/pages` - Main landing page and other page components
- `/src/app/components` - Reusable UI components and layouts
- `/src/assets` - Images and brand assets
- `/src/styles` - Global styles and theme configuration
